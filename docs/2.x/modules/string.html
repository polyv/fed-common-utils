cutStr

  • cutStr(str: string, length: number, options?: ICutStrOptions): string
    • Defined in string.ts:80

    如果目标字符串超出限制长度，则进行截断并拼接省略符号；否则返回目标字符串。 默认情况下，英文字符的单位长度为 1，非英文字符的单位长度为 2。

    example

    cutStr('测试一下', 8); // '测试一下'

    Parameters

    • str: string

      目标字符串。

    • length: number

      限制的长度。

    • Optional options: ICutStrOptions

      选项。

    Returns string

    截断后的字符串。

strLen

  • strLen(str: string, options?: number | IStrLenOptions): number
    • Defined in string.ts:43

    计算字符串长度。默认情况下，英文字符的单位长度为 1，非英文字符的单位长度为 2。

    example
    strLen('abcde;'); // 6
    strLen('abc测试；'); // 9
    strLen('abc测试；', 1); // 6
    strLen('abc测试；', { enLen: 0.5, nonEnLen: 1 }); // 4.5

    Parameters

    • str: string

      字符串。

    • Optional options: number | IStrLenOptions

      为数字时表示非英文字符单位长度（此时英文字符单位长度为 1）；为 Object 时表示选项。

    Returns number

    字符串长度。
